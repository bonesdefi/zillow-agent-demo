# Adding the Streamlit UI Screenshot

## Quick Instructions

1. Locate your screenshot file: `real-estate-ai-assistant(1).png`
2. Copy it to the docs/images directory:

```bash
cp /path/to/real-estate-ai-assistant\(1\).png docs/images/streamlit-ui-screenshot.png
```

3. Verify the file was copied:

```bash
ls -lh docs/images/streamlit-ui-screenshot.png
```

4. Commit and push:

```bash
git add docs/images/streamlit-ui-screenshot.png
git commit -m "docs: Add Streamlit UI screenshot"
git push origin main
```

The image will automatically display in:
- Main README.md (top of page)
- docs/architecture.md
- docs/deployment.md
